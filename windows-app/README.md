# LocalDrop Windows App

This folder contains the Tauri-based desktop application for LocalDrop.
It bundles a vanilla HTML/JS/CSS frontend with a Rust backend that runs an embedded HTTP server for local file streaming.

## Development

- Frontend: `src/` (Vanilla HTML/CSS/JS)
- Backend: `src-tauri/` (Rust + Axum)

For instructions on building and running, see the main repository [README](../README.md).
